# Reto de la concentración de analítica de datos
---
## Descripción del Proyecto
"Liverpool Insights" es una plataforma web desarrollada para el área legal de Liverpool enfocada en la gestión y renovación de marcas. Integra dashboards descriptivos, un buscador de marcas, un modelo de machine learning para predecir renovaciones, y “Bolo” un asistente conversacional basado en un sistema multi agente RAG capaz de consultar bases de datos relacionales y documentos legales indexados en lenguaje natural.
